Dateien zur Initialisierung der Datenbank
=========================================


Vollständiger Berliner Straßenkatalog bzw. Auschnitt für Demo::

  strassenkat_berlin.sql.gz
  strassenkat_berlin_ausschnitt.sql.gz

Weitere Straßenkataloge::

  strkatalog_braunschweig.sql.gz
  strkatalog_braunschweig_gifhorn.sql.gz
  strkatalog_gifhorn.sql.gz
  strkatalog_wolfenbuettel.sql.gz
  strkatalog_wolfsburg.sql.gz
  strkatalog_wolfsburg_gifhorn.sql.gz

Die Kombinationen werden teilweise so von den EB-Stellen
benötigt. 

Beispiel für das Format eines Straßenkatalogs im CSV-Format, wie
er in EBKuS importiert werden könnte::

  beispiel_strkatalog.csv

Beispiel für das Format von Altdaten im CSV-Format, wie sie in
EBKuS importiert werden können::

  beispiel_altdaten.csv

Gemeindeschlüssel aus ganz Deutschland. Wird ab EBKuS 4.4
importiert::

  gemeindeschluessel_in_deutschland.csv.gz

Merkmalsdefinitionen zur Initialisierung der Datenbank::

  merkmale_standard.py
